# DeepL API Web Translator

一个基于 DeepL API 的翻译 Web 页面，支持本地部署和 Docker 部署。项目只是利用官方API接口，不提供API KEY。
对于PRO用户来说使用官方页面就可以。本项目方便FREE API用户翻译文档，特点是可以自己维护术语表。
本人非程序员，只是日常工作经常使用，因此通过AI写了这个脚本满足自己工作需要。

## 安装与使用

### 本地部署

1. 复制项目到本地目录
2. 运行./run.sh，浏览器访问http://127.0.0.1:5000

### Docker部署

1. 复制项目到本地目录
2. 进目录运行docker-compose up -d
